# User-Submitted Logos

Ordered list of community-submitted concepts. These appear right after
`osm_us*.svg` in the [preview gallery](https://openhistoricalmap.github.io/ohm-creative/).

To reorder, just rearrange the bullets. To add a logo, drop a new SVG
into this folder and add its filename to this list.
