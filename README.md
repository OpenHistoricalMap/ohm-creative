# variations/

Side artifacts that are preserved but NOT part of the canonical concept set.

## folded-map-silhouette-organic.svg
A hand-feel variation of the v0.1 folded-map silhouette (used in concepts 5,
6, 7, 8, 9). Generated by jittering each polygon vertex by up to ±1.2 units
and replacing each straight edge with a very gentle Q curve.

The path data is also saved separately in `folded-map-silhouette-organic.path.txt`
for easy reuse if it gets promoted later.

## NOT canonical
The v0.1 concepts use the original crisp polygon. This file is kept as a
keepsake of an exploration round, not a replacement.
